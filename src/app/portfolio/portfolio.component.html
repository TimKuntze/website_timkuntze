Drinking game: "Ring of Fire"

This interpretation of the classic (and dangerous) drinking game Ring of Fire was built with Angular and Firebase. The following resources were applied: structural directives, dependency injection, components and bidings. After you started a new game, share the link with a friend for a multiplayer experience.

Jump 'n' Run Game: "El Pollo Loco"

This is a small JavaScript-based one-level Jump 'n' Run game named "El Pollo Loco". In this game you play the Mexican "Pepe", trying his best to stop a small, but dreadful invasion of crazy chickens ;)

Of course nothing to spend a whole night with, but anyways...give it a shot!
